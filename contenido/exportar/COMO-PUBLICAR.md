# Cómo publicar los carruseles

Cada carpeta `carrusel-NN/` tiene todo listo para postear:

- `slide-01.png … slide-NN.png` — las placas en **1080×1350** (formato 4:5, sirve para Instagram y LinkedIn).
- `carrusel-NN.pdf` — el carrusel completo en **PDF** (una página por placa).
- `post.txt` — el texto del post (caption) + hashtags, listo para copiar y pegar.

## Instagram (feed carrusel)
1. Nueva publicación → seleccioná las `slide-*.png` **en orden** (01, 02, 03…).
2. Pegá el contenido de `post.txt` como epígrafe.

## LinkedIn (documento / carrusel)
1. Crear publicación → **Documento** (ícono de hoja) → subí `carrusel-NN.pdf`.
2. Ponele un título corto al documento (ej: el tema del carrusel).
3. Pegá el contenido de `post.txt` como texto del posteo.

> El calendario sugerido (1 carrusel por semana) está en `../INDEX.md` y `../ESTRATEGIA.md`.
